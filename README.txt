VKCC SHORTENER BOT — ГОТОВЫЙ АРХИВ
=================================

Что внутри:
- main.py
- requirements.txt
- .env.example

Что умеет бот:
- отвечает на /start /help /ping /stats
- сокращает 1 ссылку сообщением
- сокращает пачку ссылок текстом
- принимает .txt и .csv файлы
- показывает прогресс обработки
- возвращает только короткие ссылки или ERROR
- ведёт простую статистику по пользователю

Как запустить:
1. Переименуй .env.example в .env
2. Вставь свой токен Telegram в строку BOT_TOKEN=
3. Установи зависимости:
   pip install -r requirements.txt
4. Запусти:
   python main.py

Пример токена:
BOT_TOKEN=1234567890:AAEXAMPLE_REAL_TELEGRAM_TOKEN_HERE

Важно:
- Этот бот НЕ использует официальный VK API для vk.cc.
- Он работает через веб-форму vk.cc.
- Поэтому 100% гарантию сокращения дать нельзя: если VK включит капчу, лимиты или изменит сайт, появятся ERROR.
- Чтобы уменьшить риск блокировки, по умолчанию стоит пауза REQUEST_DELAY_SECONDS=1.2

Рекомендации для хостинга:
- Команда запуска: python main.py
- Python 3.11+
- Если хостинг не читает .env автоматически, пропиши переменные окружения вручную в панели.
