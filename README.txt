1) Залей файлы на хостинг.
2) Добавь переменную окружения BOT_TOKEN со значением токена Telegram-бота.
3) Команда запуска: python main.py
4) Бот умеет:
   - /start
   - /help
   - /ping
   - /stats
   - одна ссылка текстом
   - много ссылок текстом
   - .txt файл со ссылками

Важно:
- Этот бот пытается сокращать ссылки через веб-форму vk.cc без официального API.
- Если VK изменит сайт, включит капчу или ограничит запросы, бот может отдавать ERROR.
- Для снижения риска увеличивай REQUEST_DELAY_SECONDS, например до 2.0 или 3.0.
